# face-cam

Install:
[Genymobile/scrcpy](https://github.com/Genymobile/scrcpy)

Run:
```sh
scrcpy --disable-screensaver --stay-awake --bit-rate 8M
```

Open [https://tnifey.github.io/face-cam/](https://tnifey.github.io/face-cam/) on the device.
Allow camera access in the browser. The browser should have full screen mode enabled for the best experience.

Everything is local, no data is sent to the server.
